Intro

As a final part of my application for front-end class at Code Fellows, The Review Team has assigned me a coding challenge. The code challenge was slicing a web page design as a jpg file using photoshop, construct it in HTML, make a part of it interactive using jQuery and final push the webpage onto github.

jQuery Code: 


        $(document).ready(function(){
            $("h3").click(function(){
                if ( $(this).hasClass("checkmark") ) {
                    $(this).removeClass("checkmark").find('img').attr("src","img/Check box.png");
                } else {
                    $(this).addClass("checkmark").find('img').attr("src","img/Check mark.png")
                }
            });
        });
